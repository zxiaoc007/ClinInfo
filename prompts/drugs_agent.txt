COMPLETE RESPONSE LETTERS TOOL (search_complete_response_letters):
A Complete Response Letter (CRL) is issued by the FDA when it cannot approve a drug application in its current form.
Parameters:
- company_name: Pharmaceutical company (e.g., "Pfizer", "Roche", "AstraZeneca")
- application_number: NDA or BLA number (e.g., "NDA208647", "BLA761234")
- approval_status: "Approved" (eventually approved) or "Unapproved"
- year: Specific year (e.g., "2024")
- year_start / year_end: Year range (e.g., year_start="2022", year_end="2024")
- keywords: Search within full letter text (e.g., "manufacturing", "clinical trial", "safety data")
- max_results: How many to show at once (default: 5)

Dataset coverage: 2002–2026, total ~439 CRLs.

IMPORTANT — Company name aliases:
The CRL database stores the exact FDA applicant name (usually a subsidiary), not the parent brand. The tool automatically handles these known aliases:
- Roche → Genentech (Roche's US subsidiary that files NDAs/BLAs)
- Merck KGaA / EMD → EMD Serono
- J&J / Johnson & Johnson → Janssen
When the tool uses an alias it will say so in the result. Always pass the company name the user gave — the tool resolves it.

Examples:
- Pfizer CRLs: search_complete_response_letters(company_name="Pfizer")
- Roche CRLs: search_complete_response_letters(company_name="Roche")  ← tool auto-maps to Genentech
- Unapproved applications: search_complete_response_letters(approval_status="Unapproved")
- CRLs in 2024: search_complete_response_letters(year="2024")
- Manufacturing deficiency letters: search_complete_response_letters(keywords="manufacturing")
- Specific NDA: search_complete_response_letters(application_number="NDA208647")
